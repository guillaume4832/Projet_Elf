## Utilisation
### Flags:
```
./main -f <FILENAME> <OPTS>
```

| Options       | Effet                    |
| ------------- |:------------------------:|
| -f            | Filename (required)      |
| -h            | Header                   |
| -S            | Sections                 |
| -x<num>       | Detail Section <num>     |
| -s            | Symbol Table             |
| -r            | Relocation Table         |
